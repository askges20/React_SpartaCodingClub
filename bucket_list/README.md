# 버킷리스트 웹 페이지 만들기
<p>- 완성된 웹 사이트 : https://sparta-react-4d6ec.firebaseapp.com/</p>

<h3>제공 기능</h3>
<p>- 버킷리스트 조회, 추가, 삭제, 완료 표시 기능</p>

<h3>사용한 기술</h3>
<p>- React 기반 웹 프로젝트</p>
<p>- Redux를 이용한 상태 관리</p>
<p>- Firebase의 Firestore Database에 데이터 저장, 관리</p>
<p>- Firebase Hosting을 이용하여 배포</p>
