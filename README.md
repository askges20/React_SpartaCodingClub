# 스파르타코딩클럽 프론트엔드의 꽃, 리액트

## 개발 환경 및 도구
- IDE : VSCode
- 언어 : JavaScript, JSX, SCSS 등
- 툴 : NPM, Yarn
- 패키지 : styled-components, react-router-dom, redux, react-redux, firebase, redux-thunk ...
- 데이터베이스 : Google Firebase Firestore
- 배포 : AWS EC2, Google Firebase Hosting
